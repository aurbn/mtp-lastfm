Installing MTP-Lastfm

If you use Ubuntu/Debian or a derivative, the preferred way of installation is the PPA repository.

Alternatively download and build the latest source from Github. You can either run from the source directory or install it using distutils. MTP-Lastfm is mostly written in Python, but has a small extension written in C to connect with the device.

To build the C extension in the src directory and run from there:
python setup.py build_ext --inplace
./mtp-lastfm

Or if you prefer to build and install (run as superuser):
python setup.py install

If you build packages for your distro let me know and I will provide a link here.

Build dependencies:

  • pkg-config
  • python-distutils-extra
  • libmtp-dev
  • python-dev

Runtime Dependencies:

  • libmtp
  • python-sqlite
  • py-gtk
  • gtk-glade
